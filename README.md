## 🎮 关于本项目

> **"用AI工具，讲一个科幻故事"**

这是一个**高中生业余时间手搓**的Galgame项目，纯做着玩，但想认真做完。

**当前状态**：主要人物尚未引入完毕，流程约**15分钟**  
**世界观**：科幻背景下的青春群像剧（具体设定随开发逐步揭露）

⚠️ **弃坑预警**：因为是学业优先的业余项目，存在弃坑可能性，但会尽量坚持更新！
